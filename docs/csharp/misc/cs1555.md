# <a name="compiler-error-cs1555"></a>编译器错误 CS1555

无法找到为“Main”方法指定的“class”  
  
 向 [**MainEntryPoint**](../language-reference/compiler-options/advanced.md#mainentrypoint-or-startupobject) 编译器选项指定了类，但在源代码中找不到该类名称。
